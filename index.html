1. Introduction to Shell Scripting

A shell script is a program written for the Unix/Linux shell. It's a text file containing a sequence of commands, which are executed by the shell. Shell scripts are essential for automating repetitive tasks, performing system administration, and integrating various tools and commands.

Key Concepts

Shell
A command-line interpreter that provides a user interface for accessing the operating system. Common shells include Bash, Zsh, and Ksh.
Shebang (`#!`)
The first line of a script, which tells the OS which interpreter to use. For Bash, this is `#!/bin/bash`.
Comments (`#`)
Lines starting with a hash symbol are ignored by the shell and are used for adding notes.

Example: "Hello, World!"

This example shows the basic structure of a script. Create a file named `hello.sh`, add the content, make it executable, and run it.

#!/bin/bash
# This is my first shell script.
echo "Hello, World!"

2. Variables

Variables store data. In shell scripting, you don't need to declare a variable's type. Define them with `NAME="Value"` (no spaces) and access them with `$NAME`.

Example: Defining and Using Variables

NAME="John Doe"
AGE=30
echo "My name is $NAME and I am $AGE years old."

Special Variables

The shell provides special variables that are very useful in scripts:

  • $0: The name of the script.
  • $1, $2: Positional arguments.
  • $#: Number of arguments.
  • $?: Exit status of last command.
  • $$: Process ID of the script.
  • $* / $@: All arguments.

3. Control Flow: `if-else`

Conditional statements allow your script to make decisions. The `if` statement evaluates a condition and executes code based on whether it's true or false.

Visual representation of an if-elif-else block.

Example: Checking a Number

#!/bin/bash
NUMBER=$1
if [ "$NUMBER" -gt 0 ]; then
    echo "The number is positive."
elif [ "$NUMBER" -lt 0 ]; then
    echo "The number is negative."
else
    echo "The number is zero."
fi

4. Loops: `for` and `while`

Loops are used to execute a block of code repeatedly. `for` loops iterate over a list of items, while `while` loops continue as long as a condition is true.

`for` Loop Example

#!/bin/bash
FRUITS="Apple Banana Cherry"
for fruit in $FRUITS; do
    echo "I like $fruit."
done

`while` Loop Example

#!/bin/bash
COUNT=1
while [ $COUNT -le 5 ]; do
    echo "Count is $COUNT"
    COUNT=$((COUNT + 1))
done

5. Functions

Functions group commands into reusable blocks, making your scripts more organized and modular. You can pass arguments to functions just like you do to scripts.

Example: A Greeting Function

#!/bin/bash
function greet_user() {
    echo "Hello, $1!"
    echo "Today's date is $(date)."
}

# Call the function with an argument
greet_user "Alice"

6. Creating & Running Scripts

Follow these three steps to bring your script to life.

  1. 1 Write the script: Use a text editor like `nano` to write your commands in a `.sh` file.
  2. 2 Set permissions: Make the file executable with `chmod +x your_script.sh`.
  3. 3 Run the script: Execute it from the current directory using `./your_script.sh`.
